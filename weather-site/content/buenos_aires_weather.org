* Datos del clima en Buenos Aires
El siguiente código permitió generar un conjunto de registros del
clima en Buenos Aires, argentina. estos datos se obtienen cada 15
minutos desde la API de openweathermap.

** Código utilizado
Para realizar este procedimiento, se empleó el siguiente código de
Python:


Definición de variables e imports necesarios:

Procesamiento de datos con fecha y hora. Además, en esta sección se
guarda los datos en un archivo .csv:

#+begin_src python :session :results output :exports both
# Unidades métricas (C° y m/s)
URL = f"https://api.openweathermap.org/data/2.5/weather?lat={LAT}&lon={LON}&appid={API_KEY}&units=metric"

try:
   
    response = requests.get(URL)
    response.raise_for_status()
    data = response.json()
    df_new = pd.json_normalize(data)
    df_new['dt'] = datetime.now().strftime('%Y-%m-%d %H:%M:%S')

    try:
        script_dir = os.path.dirname(os.path.abspath(__file__))
    except NameError:
        script_dir = os.getcwd()
        
    csv_path = os.path.join(script_dir, 'clima-buenosaires-hoy.csv')
    
    if os.path.exists(csv_path):
        df_existing = pd.read_csv(csv_path)
        df_all = pd.concat([df_existing, df_new], ignore_index=True)
        df_all.to_csv(csv_path, index=False)
    else:
        df_new.to_csv(csv_path, index=False)

    print("Datos meteorológicos de Buenos Aires agregados con éxito")

except Exception as e:
    print(f"Error en la ejecución: {e}")
#+end_src
