1. Datos del clima en Buenos Aires

El siguiente código permitió generar un conjunto de registros del clima en Buenos Aires, argentina. estos datos se obtienen cada 15 minutos desde la API de openweathermap.

1.1. Código utilizado

Para realizar este procedimiento, se empleó el siguiente código de Python:

Definición de variables e imports necesarios: 

import os
import requests
import pandas as pd
from datetime import datetime

# FUENTES CONSULTADAS:
# - OpenWeatherMap Current Weather: https://openweathermap.org/current
# - Pandas json_normalize: https://pandas.pydata.org/docs/reference/api/pandas.json_normalize.html
# - Pandas concat: https://pandas.pydata.org/docs/reference/api/pandas.concat.html

API_KEY = "62d2107136670bd7f62c2c23cea30130"
LAT = -34.6037
LON = -58.3816

Procesamiento de datos con fecha y hora. Además, en esta sección se guarda los datos en un archivo .csv: 

# Unidades métricas (C° y m/s)
URL = f"https://api.openweathermap.org/data/2.5/weather?lat={LAT}&lon={LON}&appid={API_KEY}&units=metric"

try:

    response = requests.get(URL)
    response.raise_for_status()
    data = response.json()
    df_new = pd.json_normalize(data)
    df_new['dt'] = datetime.now().strftime('%Y-%m-%d %H:%M:%S')

    try:
        script_dir = os.path.dirname(os.path.abspath(__file__))
    except NameError:
        script_dir = os.getcwd()

    csv_path = os.path.join(script_dir, 'clima-buenosaires-hoy.csv')

    if os.path.exists(csv_path):
        df_existing = pd.read_csv(csv_path)
        df_all = pd.concat([df_existing, df_new], ignore_index=True)
        df_all.to_csv(csv_path, index=False)
    else:
        df_new.to_csv(csv_path, index=False)

    print("Datos meteorológicos de Buenos Aires agregados con éxito")

except Exception as e:
    print(f"Error en la ejecución: {e}")
